Small Brief: Cleversteam search results redesign

Cleversteam outlined that they would like to have their search results redesgined, the search results had a tile based layout and can only see a few things about the aplicant. Cleversteam wanted to keep the tile layout but redesigned in a way to show more information about the applicant.

My interpretaion of the brief was that Clevesteam wanted a new way to view more information while keeping to the basic retangular tile layout. The idea was to be able to search for applicants and rather than open a new page for each evaluation they would rather be able to have the necessary information presented to them on the search page. The overal goal is to give an employer the inofmration needed to interview the applicant with saving time loading multiple pages for multiple applicants.

To redesign the results I will be using pencil and paper to create wireframes. These wireframes will act as my first ideaws for the redesign. After these are finished I will move on to create mock-up prototypes on Figma. These Figma designs will be what I intend to develop into a working search result element.
